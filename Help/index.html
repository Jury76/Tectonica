Various bits of handy knowledge

  • Summary: LocTite "Ultra Gel Control" glue seems best for PLA.
  • Summary: Acetone can be used to weld PLA parts together - just paint it onto both surfaces and press them together. Acetone cannot be used to smooth PLA like it does with ABS.
  • Summary: Friction welding PLA works pretty good.
  • Summary: When friction welding, there is some technique involved - using a right-to-left weld seems best.
